**VetConnect**

VetConnect is a telemedicine veterinary web application designed to connect pet owners, veterinarians, and administrators efficiently. It streamlines appointment booking, online consultations, prescription management, and administrative oversight within a secure and user-friendly platform.

**Features**

**User Roles:**

Pet Owner: Manage multiple pets, book appointments (in-clinic or online video calls via WebRTC), track prescriptions, view appointment history, and update profile.

Veterinarian: Approve or decline appointments, conduct consultations, and fill prescription forms during appointments.

Administrator: Oversee all users, track appointments, and monitor financial transactions.



**Authentication**: Secure JWT-based authentication for all users.

**Real-time Video Consultations:** Online appointments powered by WebRTC for seamless video calls.

**Payment Integration:** eSewa payment gateway for secure transactions.

**Appointment Management:** Full appointment lifecycle from booking to prescription delivery.


**Technologies Used**
React.js

Node.js, Express.js

MongoDB

JWT Authentication

WebRTC

eSewa Payment Gateway

RESTful APIs

Responsive Design


**Usage**

Pet owners sign up, add pets, and book appointments.

Vets manage appointment requests and conduct consultations.

Admins monitor users, appointments, and finances.
